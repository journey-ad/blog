ruby简介

ruby原指5.5级的字体格式，在美国又称Agate，它的大小正好适合注音。现在在日文中，这词不再表示字体的名称，而是表示名词形式的标音。当它被翻译回英文时，它被称为rubi（由假名译来，日文：ルビ）。但是ruby的拼法还是比较普遍。
除了字母以外，一些假名也会用ruby来标注。
　　—— 维基百科
